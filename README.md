# Power BI Reports & Dashboards

A collection of Power BI projects showcasing data analysis, visualization, and business intelligence skills.

## Projects

- *Sales Performance*: Revenue trends by region/product.  
  Features: DAX measures, drill-through, tooltips.  

- *Financial Dashboard*: Tracks P&L, expenses, and budgets.  
  Features: Dynamic titles, bookmarks, custom visuals.  

- *Customer Behavior*: Purchase patterns and segmentation.  
  Features: Cohort analysis, RLS (Row-Level Security).  

- *Operational Efficiency*: Productivity and process metrics.  
  Features: KPIs, conditional formatting.  

(More projects added over time.)

## Key Skills

- *Data Modeling*: Star schema, relationships, optimizations.  
- *DAX*: Time intelligence, iterators (SUMX, FILTER), measures.  
- *Visual Design*: Mobile layouts, hierarchies, themes.  
- *Performance*: Query folding, incremental refresh.  

## How to Use

1. Download .pbix files and open in *Power BI Desktop*.  
2. Modify datasets/DAX as needed (sample data provided).  
3. Explore or adapt for your own projects.  

> *Note*: Sensitive data is anonymized.
